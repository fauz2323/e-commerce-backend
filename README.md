## Installation

untuk melakukan instalasi silahkan clone repository ini dan jalankan perintah pada terminal

```bash
  cd nama-project
  composer Install
  cp .env.example .env // untuk linux atau macOs
  copy .env.example .env // untuk windows
  php artisan key:generate
  php artisan migrate
  npm install

```
